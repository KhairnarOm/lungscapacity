# 🫁 LungCap Prediction System

## 📌 Project Overview

The LungCap Prediction System is a Machine Learning-based healthcare project that predicts a patient's lung capacity using medical and lifestyle parameters. The project uses multiple regression algorithms to analyze patient data and provide accurate lung capacity predictions through an interactive Streamlit web application.

This project helps demonstrate how Machine Learning can be applied in the healthcare domain for predictive analysis and decision-making.

---

# 🚀 Features

* Predicts lung capacity using patient health information
* Interactive Streamlit web application
* Multiple Machine Learning algorithms implemented
* Data preprocessing and feature encoding
* Exploratory Data Analysis (EDA)
* Model comparison with accuracy evaluation
* Clean and user-friendly interface
* Real-time prediction output

---

# 🧠 Machine Learning Algorithms Used

| Algorithm                    | Accuracy (%) |
| ---------------------------- | ------------ |
| Linear Regression            | 88.08%       |
| Decision Tree Regressor      | 83.72%       |
| Random Forest Regressor      | 87.46%       |
| XGBoost Regressor            | 87.54%       |
| Support Vector Machine (SVM) | 86.88%       |

✅ **Best Performing Model:** Linear Regression

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/LungCap-Project.git
```

## 2️⃣ Navigate to Project Folder

```bash
cd LungCap-Project
```

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

## 4️⃣ Run the Streamlit Application

```bash
streamlit run LR_model.py
```
